<h1> Console CSV Reader – Python OOP Project </h1>


<b> This project demonstrates basic object-oriented programming and file I/O in Python by reading video game console data from a CSV file and storing it in a dictionary of objects. </b>

<b1> 🔹 Features

Console class with:

- Attributes: name, release year, manufacturer, generation
- is_vintage() method to check console age
- Custom __str__ method for formatted output
- CSV file parsing with error handling
- Dictionary storage using console names as keys
  
🔹 Key Concepts Demonstrated
- Class design and object instantiation
- Reading and processing CSV files
- Exception handling (FileNotFoundError)
- Dictionary data structures
- Method definitions and string formatting

This project highlights foundational Python skills in OOP, file handling, and data organization.</b1>
